## 树

链表是**特殊的树形结构**

链表是**一叉树**



```c
typedef struct Node {
    int data;
    struct Node *next[3];
} Node, *Tree;
```



`节点的数量 = 边数 + 1`

**二叉树中度为零的节点比度为二的节点多一个**



### 完全二叉树

1. 编号为i的字节点：

   左孩子编号：2 * i

   右孩子编号：2 * i + 1

2.  可以 用连续空间存储（数组）



空间优化方法 ： **记录式改计算式**



记录式：**时间复杂度小， 空间复杂度大。**

计算式：**时间复杂度小， 空间复杂度大。**
