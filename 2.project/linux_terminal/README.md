## 简易 linux 终端项目

### 项目需求

* 使用 c 语言实现简易的 linux 终端
* 能够使用 `cd`、`ls`、`touch`等简单的命令
* 配色与 linux 配色相同（ls 不考虑配色问题）



### 前期准备

1. 获取本机信息的函数
2. `chdir()` 实现 cd 命令的函数
3. `system()` 可以调用系统命令的函数
